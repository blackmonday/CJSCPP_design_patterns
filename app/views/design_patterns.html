This is an early access release that that is regularly being updated and improved as it moves closer to alpha release.

Currently each pattern is getting the following treatments:

  • Where appropriate and the coding skills are available flat example images are being replaced with coded examples along with code snippets that can be used with the GOV.UK prototype kit - very useful for user testing and rapid iteration of fully interactive clickable designs.
  • Sketch files are being replaced by more refined Sketch libraries.

With this in mind please do use the patterns and help us to iterate and improve them.

If you find any issue please contact larry.miles@hmcts.net

Design patterns

Last update on 5 June 2018

Here you will find the latest CJS Crime Programme Design Patterns for designers. These patterns can be individual 'elements' or 'components' (a collection of 'elements' used together) and every pattern include a template for Bohemia Sketch.

This is an addendum to the GDS Design Patterns and GOV.UK elements as well as informing the Single Page Platform Development Kit (PDK) to help designers to create content and prototypes for the CJS Crime Programme.

Many of these patterns include code snippets that can be used with the GOV.UK Prototype Kit.
