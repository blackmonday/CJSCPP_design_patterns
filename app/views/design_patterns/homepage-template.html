Homepage template (landing page/dashboard)

Description

This homepage (landing page/dashboard) is the typically the first page a user will see after they successfully sign in.

It is the users homepage where the content displayed will be a usefull overview of information that is important to that user to easily access items that require action. There are also options to include important links as well well as a search to find jus what the user is searching for.

There are three different templates with varying levels of complexity, which one to use will depend on the users and be determined by UX research and design.

All homepage templates use the CPP global footer; the content will be determined by user testing and UX research and design.

The main difference between the templates is the header section which is called a homepage masthead - these are detailed below.
