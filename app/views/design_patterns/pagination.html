Pagination

Description

Pagination should only be used when a large amount of records are shown. For example, if the record count is less than '50' then display all items and don't show pagination at all, if the record count is '50' or more then show pagination dispalaying up to 50 items per page.

Examples

First page

When on the first page only show the next control.

Middle pages

When there are three or more pages the middle pages will show both next and previous controls.

Last page

The last page will show the previous control only.
