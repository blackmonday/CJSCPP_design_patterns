Global navigation

Last updated on 19 February 2018

Description

This pattern is a collection of styled links that come in two variations: 1) horizontally at larger screen sizes and; 2) at smaller screen sizes the navigation changes to a stacked version that can be shown and hidden.

These links can be sectioned into groups.

View Javascript code snippet 
function globalNavShowMenu() {
    document.getElementById("globalNavShowHide").innerHTML = "Show menu &#9660;";
    document.getElementById("globalNavContent").style.display = "none";
}  
function globalNavHideMenu() {
    document.getElementById("globalNavShowHide").innerHTML = "Hide menu &#9650;";
    document.getElementById("globalNavContent").style.display = "block";
}
