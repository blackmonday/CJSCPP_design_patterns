Homepage template

(landing page/dashboard)

Last updated on 1 March 2018

Description

A homepage (landing page/dashboard) is the typically the first page a user will see after they successfully sign in.

This is where the content displayed will be a usefull overview of information that is important to that user to easily access items that require action. There are also options to include important links as well well as a search to find jus what the user is searching for.

There are three different templates with varying levels of complexity, which one to use will depend on the users and be determined by UX research and design.

All homepage templates use the CPP global footer; the content will be determined by user testing and UX research and design.

The main difference between the templates is the header section which is called a homepage masthead - these are detailed below.

How do I use these templates?

Coded versions of these templates have been created for use with the GOV.UK prototype kit:

homepage-templates.zip

This ZIP archive includes the following files:

  • Specific to template 1
    • 'homepage-template-1.html' - this is the main page template and should be placed in your 'views' folder (prototype folder > app > views) - you can edit the content in this file as needed
    • 'homepage_header_template_1.html' - this is an include for the masthead and should be copied to your prototypes 'includes' folder (prototype folder > app > views > includes) - you can edit the content in this file as needed
  • Specific to template 2
    • 'homepage-template-2.html' - this is the main page template and should be placed in your 'views' folder (prototype folder > app > views) - you can edit the content in this file as needed
    • 'homepage_header_template_2.html' - this is an include for the masthead and should be copied to your prototypes 'includes' folder (prototype folder > app > views > includes) - you can edit the content in this file as needed
  • Specific to template 3
    • 'homepage-template-3.html' - this is the main page template and should be placed in your 'views' folder (prototype folder > app > views) - you can edit the content in this file as needed
    • 'homepage_header_template_3.html' - this is an include for the masthead and should be copied to your prototypes 'includes' folder (prototype folder > app > views > includes) - you can edit the content in this file as needed
  • Specific to all homepage templates
    • 'homepage_header_template_black_bar.html' - this is an include for the black bar at the very top of the page and should be placed in your projects 'includes' folder (prototype folder > app > views > includes)
    • 'layout-dashboard.html' - this should be placed in your projects 'views' folder (prototype folder > app > views)
    • 'CPP_template_homepage.html' - This should be placed in your prototype folders 'lib' folder (prototype folder > lib)
