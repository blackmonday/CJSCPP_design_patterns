Form elements: Type-ahead

Last updated on 17 April 2018

Description

When typing text into a form field with specific items that can be selected all related items that match the text being typed will be displayed in real-time.

Examples

Example one

This is how the empty form field looks before any text has been typed.

Example two

This shows the form element in focus and even before any characters are typed the type-ahead pattern will be display all items.

Example three

As characters are typed the number of items listed is refined to show only the items that contain the characters typed.

In this example 4 items are visible and if there are more items the user can scroll to view them and resize the height of the list.

Example four

This shows the form field populated with the item selected.

Example five

If no results are found the message 'No results found' is displayed.
