Summary item

Last updated on 23 February 2018

Description

This pattern can be used to summarise at a high level important information.

Examples

Small variants

8

statements

48px

16px

Summary items can link to more content or be text only.

Large variants

44

exhibits

80px

16px

Summary items can link to more content or be text only.

In use

This is an example of the large variant in use.

Code snippets

View CSS code snippet 
/* Summary item pattern */
.summary-item-small h2 {
    margin-bottom: 0;
    font-size: 48px;
    line-height: 36px;
    font-weight: bold;
}
.summary-item-large h2 {
    margin-bottom: 0;
    font-size: 80px;
    line-height:74px;
    font-weight: bold;
}
.summary-item-small a, .summary-item-large a {
    text-decoration: none;
}
.summary-item-small p, .summary-item-large p {
    margin-top: 0;
    margin-bottom: 0;
    font-size: 16px;
    line-height: 36px;
    font-weight: bold;
}

    </a>
</div>

Additional information

  • always left aligned with grid columns
  • never combine the small and large variants
  • large text is always bold
  • small text can be bold or light
  • links use standard link colours - note that links are NOT underlined
