Buttons

Last updated on 25 January 2018

Description

We use a combination of GOV.UK and our own CPP styled buttons.

Use buttons to move through a transaction, aim to use only one button per page where possible.

Code snippet

CSS

.button-secondary {
     @include button($grey-3);
}

.button-tertiary {
     @include button($red);
}

HTML

<input class="button-primary" type="submit" value="Primary (GOV.UK)">

<input class="button-secondary" type="submit" value="Secondary(CPP)">

<input class="button-tertiary" type="submit" value="Tertiary (CPP)">

<a class="button button-start" href="#" role="button">Start now</a>

This code snippet can be used with the GOV.UK Prototype Kit.
