Label

Last updated on 25 January 2018

Description

The label pattern is used to display important details or information related to a specific item, for example, a case that is awaiting plea.

Example

Awaiting plea

Code snippet

CSS

.label {
   border: 2px solid #0B0C0C;
   padding: 6px 9px 3px 9px;
   font-size: 19px;
   font-weight: bold;
   text-transform: uppercase;
}

HTML

<span class="label">AWAITING PLEA</span>

This code snippet can be used with the GOV.UK Prototype Kit.
