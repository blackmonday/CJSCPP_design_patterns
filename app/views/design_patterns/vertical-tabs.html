Vertical tabs

Description

Use vertical tabs to display a lot of content in a smaller space to avoid very long scrolling pages.

There is also a horizontal tab pattern here.

Additional information

Typography

The text in the tab is repeated as a header in the tab content.

Linking text is not underlined.

Tick icon

The tick icon is green (#006435) and 14 x 11 pixels.
