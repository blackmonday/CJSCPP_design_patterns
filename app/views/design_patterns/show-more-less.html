Show more/less

Last updated on 23 February 2018

Description

When a text block is truncated a show more link is displayed at the end of the truncated text.

This text link will change to a show less link at the end of the text block.

View Javascript code snippet 
/* Show more/less pattern */
function showMoreLess() {
    var x = document.getElementById("showMoreLessContent");
    y = document.getElementById("showMoreLessControl");
    if (x.style.display === "inline") {
        x.style.display = "none";
        y.innerText = "More";
    } else {
        x.style.display = "inline";
        y.innerText = "Less";
    }
}
