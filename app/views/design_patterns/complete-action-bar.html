Complete action bar

Last updated on 13 February 2018

Description

The complete action bar is a component pattern, incorporatin the 'status indicator' pattern, used to indicate that status of and/or additional information for a thing

ECHR doesn't apply
Saved
Change

This example shows includes the status indicator pattern - saved variation.

View CSS code snippet 
/* Complete action bar */
.complete-action-bar div.first {
    float: left;
}
.complete-action-bar div.second {
    float: right;
}
.complete-action-bar-link {
    font-size: 16px;
    text-decoration: none;
    padding-left:10px;
}

Additional information

If an item has not been shared or saved then there is not status indicator displayed.

The status indicator for "Saved" is displayed temporarily when and item is saved and after a short while it changes to the status indicator with a grey tick icon and no text.
