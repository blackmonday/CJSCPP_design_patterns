Search (element)

Description

The search component is based on the GOV.UK standard search box and is comprised of the following:

  • search input
  • search result inset text

This is a typical search box.

This is shown with optional hint text.

The search component follows the standard GOV.UK styling as shown in the exaples and as follows:

Default button colour

 
#005EA5
$link-colour

Hover button colour

 
#004F8C

Left border is 10px wide and the colour is:

 
#BFC1C3
$border-colour

There is 15px space between left border and text.

How search results are displayed is not fixed or predefined - it should be appropriate for the user need it is addressing.
