Global header (CPP)

Last updated on 2 March 2018

Description

The majority of the common platform currently uses this CPP specific global header.

This page is using the CPP global header and global footer.

How do I use this?

You will need the following file and code (these are the same for both the CPP global header and footer):

  • layout-CPP.html - place this in your prototypes 'views' folder (prototype folder > app > views)
  • CPP-template.html - place this in your prototypes 'lib' folder (prototype folder > lib)
Place this code at the top of your page template: 
{% extends "layout-CPP.html" %}

This replaces the default code {% extends "layout-CPP.html" %}

When a user is NOT signed in the header will display the service name to the left, IdAM sign in link to the right.
